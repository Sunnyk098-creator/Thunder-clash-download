Thunder Clash is a dynamic, feature-rich gaming platform designed for competitive players. The app offers a seamless and engaging experience with the following key features:

  • Electrifying Design: Features a dynamic glassmorphism UI with electric blue/cyan accents, animated text glows, and a thunderbolt background emoji.
  • Splash Screen: Greets users with an animated logo, app name, loading spinner, and developer credit on startup.
  • Server Connection: Allows users to connect to various game servers, view their online/offline status, and enter server-specific passwords and activation codes.
  • User Authentication: Supports secure user login and sign-up with unique @usernames and a 4-digit Thunder PIN for transactions.
  • Account Management: Users can update their password and log out of their account.
  • Banned Account Handling: Provides a dedicated screen for banned users to request an account review from administrators.
  • Maintenance Mode: Displays a maintenance screen when the app or server is undergoing updates.
  • Home Dashboard: Presents a personalized welcome message, current wallet balance, and quick action buttons for sending money and viewing pending payments.
  • Game Selection: Offers a grid-based selection for various game types, including "File Work", "Free Fire", "Ludo", and "Mystery Box", to browse specific matches.
  • Live Match Listings: Displays live matches with details like time, prize pool, and entry fee. Users can join matches by providing their respective game IDs (e.g., Free Fire UID, WhatsApp number).
  • Match Details: Provides in-depth information for joined matches and allows users to submit a "Finish Match" request.
  • Secure Wallet System: Manages user balances, enables sending money to other users via Thunder ID with PIN verification, and facilitates withdrawal requests via UPI ID.
  • Transaction History: Keeps a detailed record of all withdrawals, sent/received funds, and match-related transactions (entry fees, winnings).
  • Pending Payments: Shows a dedicated screen for pending balance and a history of incoming and outgoing pending transactions.
  • Personalized Themes: Users can customize the app's accent color theme, with exclusive "Premium" themes available for premium subscribers.
  • Premium Membership: Offers an ad-free experience and access to exclusive app themes for premium users, with a special welcome notification upon activation.
  • Admin Notices: Notifies users of important announcements or messages from the administrator.
  • In-App Advertising: Displays advertisements for non-premium users.
  • Background Music & Sound Effects: Enhances user experience with ambient background music and click sounds.
  • Profile Page: Centralizes user information and provides options to change password and find support contact (Telegram link).
